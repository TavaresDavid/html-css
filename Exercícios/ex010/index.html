Observações

Para adicionarmos um link no nosso site, usaremos a tag (a). Em seguida ela automaticamente vai pedir a (href) que é a referência ou seja, o link desejado.


DETALHE: o link quando adicionado com essa tag, ao clicar, ele não voltará ao site atual.

Ainda nessa tag, podemos fazer com que nosso link quando clicado no site, siga para uma nova página sendo possivel voltar no site atual. Para fazer isso acrescentaremos na tag: (target= "blank" rel= "external") ficando a tag completa da seguinte forma: (a href="nosso link" target="_blank" rel="external")
