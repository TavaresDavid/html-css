Usando a tag blockquote, podemos citar determinado trecho de um texto/livro, usando as formatações padrão de um livro ou de um artigo escolhido sendo adicionado automaticamente pela mesma tag.

Ainda nesta mesma tag, podemos adicionar outra sendo (cite=) e logo colar uma determinado link do artigo/livro ficando assim : (blockquote cite=" link desejado"), mas, este não irá aperecer no site servindo apenas pra que o sistema entenda que esse trecho colado se refere a uma tipo de site o autor específico.
