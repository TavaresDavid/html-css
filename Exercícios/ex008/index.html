Principais Formatações

Negrito / Destaque

Nessa frase termo em negrito usando a tag (B) (não semântica).

Nesta frase, tenho um termo em destaque usando a tag (STRONG) (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag (i) (não semântica).

Nesta frase, temos um termo em ênfase usando a tag (EM) (semântica).

Texto marcado

Podemos criar também uma texto maracado usando a tag (MARK).

E no outro parágrafo, temos outro texto marcado no final.

Podemos modificar a cor da marcação como mostrado acima. Faremos de forma idêntica ao modelo a seguir MARK STYLE= "BACKGROUND-COLOR: então basta escolhermos a cor e selecioná-la. "

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse prágrafo.

Para criar um texto grande usaremos a tag BIG e para texto pequeno, usaremos a tag SMALL.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado

Para fazermos isto, usaremos a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Para fazermos isto vamos usar a tag INS (sendo semântica).

Existe também o Sublinhado com a tag U que é (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3

Basta usarmos a tag SUP (sendo assim selecionaremos somente o que queremos deixar como elevado) ps: SUP é sempre pra cima.

Texto subscrito

Para inserir coisas do tipo H2O

Basta usarmos a tag SUB (sendo assim selecionaremos somente o que queremos descer) ps: SUB é sempre pra baixo.
